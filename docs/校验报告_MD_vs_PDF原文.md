## 校验方法论

对MD文件中的12个算法模块及3个附录，逐一检索PDF原文中的对应定义/描述，核实：
1. **定义准确性**：算法语言是否忠实反映原文语义
2. **出处准确性**：标注的课号是否准确
3. **有无主观添加**：是否存在超出原文的逻辑或参数
4. **有无关键遗漏**：是否遗漏原文中的重要约束条件

校验共进行6轮，逐轮收紧查验粒度：
- 第1轮：核心术语与模块出处校验

- 第3轮：原文逐词对照 + 逻辑完整性

- 第5轮：全面终查确认收敛
