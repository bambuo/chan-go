# 买卖点识别算法

> 本文遵循 [中文伪代码规范](./中文伪代码规范.md)。

### 第一类买点

**条件**

1. 处于下跌趋势（至少有两个向下的中枢）。
2. 价格处于最后一个中枢下方。
3. 出现底背驰。

**识别方法**

从最近的中枢开始，向前遍历每个中枢。对每个方向向下的中枢，在该中枢下方寻找底背驰信号。若找到类型为底背驰的信号，则确认该位置为第一类买点。

### 第二类买点

**条件**

1. 第一类买点已经出现。
2. 第一类买点之后出现第一次反弹。
3. 第一次反弹之后的回调不创新低（即回调低点高于第一类买点的价格）。

**识别方法**

从第一类买点出发，找到之后的第一次反弹笔，再找到该反弹之后的第一次回调笔。若回调笔的低点高于第一类买点的价格，则该回调低点为第二类买点。

### 第三类买点

**条件**

1. 有一笔离开中枢（向上突破）。
2. 离开后的第一次回调不触及中枢上沿（ZG）。

**识别方法**

找到离开中枢的那一笔，再找到该笔之后的第一笔回调。若回调笔的最高价低于中枢上沿，则进一步检查回调笔内部是否出现次级别的底背驰。若底背驰存在，则确认该回调的低点为第三类买点。

### 卖点（对称关系）

- **第一类卖点**：上涨趋势最后一个中枢上方 + 顶背驰。
- **第二类卖点**：第一类卖点之后，第一次回抽不创新高。
- **第三类卖点**：向下离开中枢后，第一次回升不触及中枢下沿。
