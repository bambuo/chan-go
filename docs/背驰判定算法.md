# 背驰判定算法

> 本文遵循 [中文伪代码规范](./中文伪代码规范.md)。

### 输入

- `价格极值[]`：价格高点序列（或低点序列）。
- `MACD柱[]`：MACD柱状图（红柱/绿柱）。

> 背驰判定的两种方法：
> 1. **MACD柱面积法**（第24课）—— 最稳定，推荐用于程序化实现。
> 2. **均线吻面积法**（第15课）—— 前后两个同向"吻"之间短期/长期均线相交围成的面积，后面积 < 前面积即为背驰。改进：**缠中说禅趋势平均力度** = 趋势力度 ÷ 吻的持续时间，可提前判断。

### 算法步骤

依次比较每一对相邻的价格极值。

**顶背驰检查**

若当前极值高于前一个极值（价格创新高），则检查 MACD 柱状图面积是否同步创新高：

- 计算当前笔对应的 MACD 柱面积（取绝对值之和）。
- 计算前一笔对应的 MACD 柱面积（取绝对值之和）。
- 若当前面积 < 前面积 × 0.95（面积缩小 5% 以上），则判定为顶背驰。

记录的信息包括：背驰位置、当前高点、前高、MACD 面积比。

**底背驰检查**

若当前极值低于前一个极值（价格创新低），则同样比较两段的 MACD 柱面积。若当前面积 < 前面积 × 0.95，判定为底背驰。

### 盘整背驰判定

盘整中的背驰是指离开中枢的笔与进入中枢的笔之间的 MACD 面积比较：

若离开笔的 MACD 面积 < 进入笔的 MACD 面积 × 0.95，则判定为盘整背驰。

### 区间套算法（精确定位）

从高级别背驰点开始，逐级向下找精确转折点：

已在高级别确认一个背驰点 → 获取该背驰点所处的时间范围 → 在次级别数据的时间范围内检测背驰 → 取最接近范围末尾的背驰点 → 将范围缩小到该点 → 继续向下一个级别重复。最后一个级别的背驰点即为精确转折点。
