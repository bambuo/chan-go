# 缠论核心可操作交易点分类清单

> 基于《缠中说禅：教你炒股票108课》原文整理，按可程序化程度由高到低分级排列。

---

## 一、完全可程序化的机械规则（确定性操作）

### 1.1 K线包含处理
| 要素 | 内容 |
|------|------|
| **触发条件** | 相邻两根K线存在包含关系 (H_a≤H_b ∧ L_a≥L_b) ∨ (H_a≥H_b ∧ L_a≤L_b) |
| **操作规则** | 向上处理：取高高（MAX(H), MAX(L)）；向下处理：取低低（MIN(H), MIN(L)） |
| **方向判定** | 以前一根K线为基准，当前K线H更高→向上，L更低→向下 |
| **出处** | 第62课：分型、笔与线段 |
| **可确定性** | ⭐⭐⭐⭐⭐ 纯机械 |

### 1.2 分型识别
| 要素 | 内容 |
|------|------|
| **顶分型** | 中间K线最高点 > 左右两根最高点 ∧ 中间K线最低点 > 左右两根最低点 |
| **底分型** | 中间K线最低点 < 左右两根最低点 ∧ 中间K线最高点 < 左右两根最高点 |
| **确认条件** | ⚠️ 实践补充（非原文明确定义）：顶分型后需后续K线跌破第三根K线低点；底分型后需后续K线升破第三根K线高点，否则可暂不生效 |
| **出处** | 第62课：分型、笔与线段 |
| **可确定性** | ⭐⭐⭐⭐⭐ 纯机械 |

### 1.3 笔的划分
| 要素 | 内容 |
|------|------|
| **定义** | 连接一个底分型和一个顶分型（或反之），中间至少1根独立K线，且顶分型最高K线区间须至少部分高于底分型最低K线区间（第77课区间条件） |
| **步骤** | ①确定全部标准分型 → ②同性质分型去重（顶取高的、底取低的）→ ③相邻顶底交替连接 + K线区间检查 |
| **新笔概念** | ⚠️ 社区延伸概念（非原文术语），原文第77课未使用"新笔"一词 |
| **出处** | 第62课、第65课、第77课 |
| **可确定性** | ⭐⭐⭐⭐⭐ 纯机械 |

### 1.4 中枢识别
| 要素 | 内容 |
|------|------|
| **定义** | 至少三个连续次级别走势类型的重叠区间 = (max(a2,b2,c2), min(a1,b1,c1)) |
| **延伸条件** | 后续走势继续与该区间重叠（围绕中枢震荡） |
| **扩展条件** | 两个同级别中枢的波动区间有重叠 → 升级为高级别中枢 |
| **出处** | 第17课：走势终完美；第18课；第20课 |
| **可确定性** | ⭐⭐⭐⭐⭐ 纯机械 |

### 1.5 走势类型分类
| 要素 | 内容 |
|------|------|
| **盘整** | 某级别的走势中只包含一个中枢 |
| **趋势（上涨）** | 至少两个依次同向的中枢，方向向上，中枢区间不能有重叠 |
| **趋势（下跌）** | 至少两个依次同向的中枢，方向向下，中枢区间不能有重叠 |
| **出处** | 第17课：走势终完美；第18课 |
| **可确定性** | ⭐⭐⭐⭐⭐ 可回溯确定，当下须确认中枢数量 |

### 1.6 线段划分
| 要素 | 内容 |
|------|------|
| **基本条件** | 至少由3笔组成，开始三笔必须有重合 |
| **第一种破坏** | 新方向相反的第一笔直接笔破坏前一段 → 特征序列出现分型确认（第78课：笔破坏） |
| **第二种破坏** | 第一笔未笔破坏前段 → 特征序列第一、二元素间有缺口 → 需第二特征序列确认分型（第67课缺口分类 + 第78课第二特征序列） |
| **出处** | 第67课：线段的划分标准；第71课；第78课 |
| ⚠️ 注意 | 第67课与第78课有**两种不同的"两种情况"分类**：前者按特征序列缺口分，后者按笔破坏与否分，不可混淆 |
| **可确定性** | ⭐⭐⭐⭐ 大部分可确定，极端情况需等确认 |

## 二、可数值化判定的交易信号（高确定性）

### 2.1 背驰判定（MACD辅助）
| 要素 | 内容 |
|------|------|
| **趋势背驰** | 两个同向中枢背驰段：后段MACD柱面积 < 前段MACD柱面积 |
| **盘整背驰** | 离开中枢的笔 vs 进入中枢的笔，MACD面积缩小 |
| **判断工具** | MACD柱面积（最稳定）、黄白线高度、趋势力度 = 均线吻面积 |
| **重要原则** | ⚠️ "没有趋势，没有背驰"——盘整中无所谓背驰 |
| **出处** | 第15课：没有趋势没有背驰；第24课：MACD对背驰的辅助判断；第27课：盘整背驰 |
| **可确定性** | ⭐⭐⭐⭐ 数值比较可程序化，但需确认走势类型 |

### 2.2 区间套精确定位
| 要素 | 内容 |
|------|------|
| **方法** | 从大级别背驰段逐级向下找：日线→30分钟→5分钟→1分钟 |
| **目标** | 将大级别转折点精确到最低级别的一笔 |
| **出处** | 第61课：区间套定位标准图解 |
| **可确定性** | ⭐⭐⭐⭐ 递归逻辑可程序化 |

## 三、三类买卖点（核心交易信号）

### 3.1 第一类买点
| 要素 | 内容 |
|------|------|
| **条件** | 下跌趋势（≥2个向下中枢）+ 最后一个中枢下方 + 底背驰确认 |
| **安全依据** | 下跌完成后必转为盘整或上涨（走势终完美） |
| **出处** | 第17课：走势终完美 |
| **可确定性** | ⭐⭐⭐ 背驰确认需等待，当下有一定模糊性 |

### 3.2 第二类买点
| 要素 | 内容 |
|------|------|
| **条件** | 第一类买点之后，第一次次级别回调不创新低，在上一买点之上企稳 |
| **安全依据** | 上涨或盘整都至少有三个次级别运动，回调后必有新的一波 |
| **出处** | 第17课：走势终完美 |
| **可确定性** | ⭐⭐⭐⭐ 条件明确，回调完成即可确认 |

### 3.3 第三类买点的系统化操作策略
| 要素 | 内容 |
|------|------|
| **条件** | 次级别离开中枢 → 次级别回试不触及中枢上沿(ZG) → 出现次级别底背驰/企稳 |
| **最佳时机** | 只买第一个中枢的第三类买点 |
| **卖出条件** | ①次级别不能新高或盘整背驰 → 坚决卖；②该级别出现背驰 → 卖一半；③继续无背驰 → 持有 |
| **出处** | 第20课：中枢级别扩张及第三类买卖点；第49课：利润率最大操作模式 |
| **可确定性** | ⭐⭐⭐⭐ 条件明确，是三买中最适合程序化的 |

### 3.4 三类卖点（反之）
| 类型 | 条件 |
|------|------|
| **第一类卖点** | 上涨趋势最后一个中枢上方 + 顶背驰 |
| **第二类卖点** | 第一类卖点后第一次回抽不创新高 |
| **第三类卖点** | 中枢下方第一次回升不触及中枢下沿(ZD) |

## 四、辅助操作策略（部分需人工判断）

### 4.1 分型操作策略
| 要素 | 内容 |
|------|------|
| **底分型买入** | 确认底分型 → 切换次级别找二买 → 介入 |
| **顶分型卖出** | 顶分型当天冲高卖出 → 确认是否有效跌破5均线 → 假跌破（分型延续）则在小级别盘整背驰回补 |
| **出处** | 第79课：分型的辅助操作；第82课：分型结构的心理因素 |
| **可确定性** | ⭐⭐⭐ 分型的延续/转折需结合小级别判断 |

### 4.2 均线"吻"系统
| 类型 | 定义 |
|------|------|
| **飞吻** | 短期均线略略走平后继续按原来趋势进行下去 |
| **唇吻** | 短期均线靠近长期均线但不跌破或升破，然后按原来趋势继续下去 |
| **湿吻** | 短期均线跌破或升破长期均线甚至出现反复缠绕，如胶似漆 |
| **"中继"vs"转折"** | ⚠️ PDF中"中继"仅用于均线缠绕的延续/转折判断（第11课），**不用于分型** |
| **出处** | 第11课：不会吻，无以高潮 |
| **可确定性** | ⭐⭐⭐ 均线系统辅助性强，独立操作性一般 |

### 4.3 板块强弱轮动
| 要素 | 内容 |
|------|------|
| **计算方法** | 板块中收盘价在N日均线之上的股票数 / 总股票数 × 100 |
| **参考均线** | 34日均线（斐波那契参数之一：5,13,21,34,55,89,144,233） |
| **用法** | 轮动操作时，优先选强度比值上升的板块 |
| **出处** | 第106课：均线、轮动与缠中说禅板块强弱指标 |
| **可确定性** | ⭐⭐⭐⭐ 纯数值计算 |

## 五、级别与递归体系（框架性概念）

### 5.1 级别递归
| 要素 | 内容 |
|------|------|
| **核心思想** | 级别是递归出来的，不是时间决定的 |
| **递归链** | K线 → 分型 → 笔 → 线段 → 最小中枢 → 更高级别中枢 → ... |
| **经验对应** | ⚠️ 1分钟→笔, 5分钟→线段, 30分钟→中枢, 日线→高一级, 周线→更高一级（仅经验参考） |
| **出处** | 第17课、第19课、第35课 |
| **可确定性** | ⭐⭐⭐ 递归框架确定，具体级别边界有弹性 |

## 可操作性分级总览

| 等级 | 信号/规则 | 程序化难度 |
|------|----------|-----------|
| 🟢 A级 | K线包含处理、分型识别、笔划分、中枢区间计算、板块强弱 | 可直接编码 |
| 🟡 B级 | 线段划分、走势类型分类、均线吻系统、背驰MACD面积比较、区间套 | 可编码，边界条件需测试调参 |
| 🟠 C级 | 第一类买卖点、第三类买卖点、分型操作策略 | 可编码，需等待信号确认 |
| 🔴 D级 | 延续/转折判断、趋势延伸判断、背驰级别转换 | 部分需人工辅助 |
