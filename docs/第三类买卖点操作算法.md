# 第三类买卖点操作算法

> 本文遵循 [中文伪代码规范](./中文伪代码规范.md)。

### 第三类买点的系统化操作策略

**选股**

遍历股票池，对每只股票在指定级别（如 30 分钟）上找到最新中枢，检查是否出现了第三类买点。只介入上涨趋势中第一个中枢的第三类买点。

**买入**

在次级别走势的回调中寻找底背驰，在底背驰确认的价位入场。

**卖出（三种情况）**

- **情况一**：次级别向上反弹不能创新高，或出现盘整背驰 → 全部卖出。
- **情况二**：当前级别出现背驰 → 卖出至少一半仓位。
- **情况三**：继续向上且无背驰 → 持有，等待情况一或情况二发生。

### 风险控制规则

- **规则一**：买错的成本远比卖错高，确认买错须立即止损。
- **规则二**：价格跌破入场价 5% 以上 → 确认买错，清仓止损。
- **规则三**：次级别出现第二类卖点且价格跌破该卖点 → 退出。
- **规则四**：次级别出现第三类卖点且确认反转 → 退出。
